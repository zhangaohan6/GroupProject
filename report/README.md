# Report (CVPR format — required)

The course requires the **official CVPR LaTeX template**; a wrong template can score 0.

1. Download the CVPR template (linked in the course spec / from CVPR author kit) and drop
   these into this folder: `cvpr.sty`, `ieeenat_fullname.bst`, and any files it needs.
2. Compile `main.tex` (Overleaf: upload this folder; or `pdflatex main && bibtex main && pdflatex main x2`).
3. `main.tex` already has all rubric-mapped sections + `\usepackage[pagenumbers]{cvpr}`.
   `refs.bib` seeded with the required citations. Keep main text <= 10 pages (refs excluded).

Recommended: use Overleaf and share with the group (5 co-authors editing).
